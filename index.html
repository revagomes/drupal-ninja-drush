Taijutsu

download (dl)

Baixa o core do Drupal e projetos contribuídos.

enable (en)

Ativa um ou mais módulos.

disable (dis)

Desativa um ou mais módulos.

Taijutsu

cache clear (cc)

Limpa todos os caches.

uninstall (pm-uninstall)

Desinstala um ou mais módulos.

update (up)

Atualiza o código do projeto e aplica as atualizações de BD caso sejam necessárias (update.php)

Taijutsu

status (st)

Lista informações básicas a respeito da instalação do Drupal.

updatecode (upc)

Atualiza apenas o código do projeto. Move os arquivos antigos para o diretório de BKP

updatedb (updb)

Executa o processo de update.php vi linha de comando.

Kenjutsu

variable get (vget)

Retorna uma lista de uma ou todas as variáveis do site com seus respectivos valores.

variable set (vset)

Atribui valor a uma variável.

variable delete (vdel)

Deleta uma variável.

Choho

watchdog delete (wd)

Deleta todas as Mensagens ou apenas as de um determinado tipo especificado.

watchdog show (ws)

Apresenta a mensagens de log do watchdog mais recentes. Permite o filtro por tipo da mensagem.

Shurikenjutsu

sql-cli (sqlc)

Abre uma interface SQL em linha de comando usando as credenciais do configuradas no settings.php do Drupal.

sql-query (sqlq)

Executa uma query no banco de dados.
